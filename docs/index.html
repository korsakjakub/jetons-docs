Get Started

How It Works

1. Host a Game

Open the Jetons app, choose Cash Game or Tournament, set the blinds and starting stack, and tap Create Game.

2. Players Join

Players scan the QR code or open the URL shown on screen. They can add a selfie for their avatar, then join the table from any browser.

3. Play

Host deals. Players post blinds from their own phones. Everyone bets in turn.

Quick Setup

  • Host device: Android phone running the Jetons app
  • Player devices: Any browser on the same Wi-Fi network
  • No internet required: Everything runs locally
