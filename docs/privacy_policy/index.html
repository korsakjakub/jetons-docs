  • Contact: korsakxyz@gmail.com

1. Introduction

Jetons is a local-network poker chip manager for Android. This privacy policy explains what information is collected when you use the app, how it is used, and what choices you have.

2. Information We Collect

2a. Data stored locally on your device

The following information is stored only on the Android device running the app and is never transmitted to external servers:

  • Player names — entered by each player when joining a game session.
  • Avatar photos (selfies) — optionally captured via the device camera. Stored locally on the host device only and shared with other players on the same local Wi-Fi network during an active game session.
  • Game history — hand records, scores, and settings stored in a local SQLite database for replay and history functionality.

2b. Anonymous crash and diagnostic data

To improve stability and fix bugs, the app uses Firebase Crashlytics (provided by Google LLC) to automatically collect anonymous crash reports. This data includes:

  • Stack traces and error messages at the time of a crash
  • Device model, manufacturer, and Android OS version
  • App version and build configuration
  • App state at the time of the crash (e.g. which screen was active)
  • Basic performance metrics (e.g. app startup time)

This data does not include:

  • Player names, avatar photos, or any game content
  • Your identity, email address, or Google account
  • Device location or IP address
  • Advertising identifiers

2c. Planned future features (not yet active)

The following features are planned but not yet implemented. This policy will be updated with full details before any of these features are enabled:

  • Google Play Games Services — optional sign-in for leaderboards and achievements.
  • Cloud storage — saving game sessions and avatar images to a remote server for backup and cross-device access.

3. How We Use Information

Data Purpose
Player names Displayed during gameplay on all devices in the session
Avatar photos Displayed as player avatars during gameplay
Game history Local history and replay functionality
Crash reports Diagnosing and fixing bugs to improve stability

4. Data Sharing and Third Parties

We do not sell, trade, or transfer personal information to any third party.

  • Firebase Crashlytics (Google LLC) — anonymous crash and diagnostic data is transmitted securely to Google’s servers for processing. This is subject to Google’s Privacy Policy and the Firebase Terms of Service.
  • Other players on your local network — during an active game session, player names and avatar photos are visible to all devices connected to the same local Wi-Fi network via the app’s embedded web server. No data leaves your local network.
  • No other third parties receive any data from this app at this time.

5. Data Storage and Security

  • All game data (player names, avatars, history) is stored locally on your Android device.
  • Local network communication (between the host device and player devices) occurs only within your local Wi-Fi network and is not transmitted to the internet.
  • Crash report data is transmitted to Google via encrypted HTTPS connections.
  • Future cloud features will use encrypted connections and industry-standard security practices.

6. Your Choices

Local data

All locally stored data (player names, avatars, game history) can be permanently deleted by uninstalling the app or clearing app data via Android Settings > Apps > Jetons > Storage > Clear Data.

Crash reporting

You can opt out of anonymous diagnostic data collection via Android Settings > Google > Usage & diagnostics (the exact path may vary by device and Android version).

Future cloud data

When cloud features are introduced, you will be able to request deletion of your cloud data. Details will be provided when those features launch.

7. Children’s Privacy

Jetons is not directed at children under the age of 13. We do not knowingly collect personal information from children under 13. If you believe a child under 13 has provided personal information through the app, please contact us at korsakxyz@gmail.com and we will take steps to delete such information.

  • Material changes will be communicated within the app or via the Play Store listing.
  • Continued use of the app after changes have been posted constitutes acceptance of the updated policy.

We encourage you to review this policy periodically.
