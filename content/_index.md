## Free vs Premium

Jetons is free to use. Here's what you get:

| Feature | Free | Premium |
|---------|:----:|:-------:|
| Cash Game & Tournament modes | ✓ | ✓ |
| Browser-based player join | ✓ | ✓ |
| QR code & URL sharing | ✓ | ✓ |
| Player avatars & selfies | ✓ | ✓ |
| Game history & replay | ✓ | ✓ |
| Hands per hour | 30 | Unlimited |
| Observer view | — | ✓ |

**The free version is enough** for casual home games. The 30 hands/hour limit is generous — at ~2 minutes per hand, that's 60+ minutes of play before the limit kicks in.

**Premium** is a one-time purchase that removes the limit and enables the observer view — great for putting the game on a TV for spectators, or for longer tournament sessions.
